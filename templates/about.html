About This Chatroom Project

Introduction

This is a simple chatroom project built with Flask, Python, and MySQL. It allows users to send and receive messages in real-time, providing a platform for casual communication.

How It Works

Users can enter messages in the input field, which are then stored in a MySQL database. The messages are displayed in a dynamic list, and new messages are shown in real-time using AJAX.

  • Message history is saved and displayed when the page is refreshed.
  • New messages are dynamically added to the chat window without refreshing the page.
  • Users can interact with the chatroom instantly.

Project Goals

The main goals of this project are to:

  • Learn how to build a web application using Flask and MySQL.
  • Understand how to implement real-time updates using AJAX.
  • Build a user-friendly and responsive interface.
